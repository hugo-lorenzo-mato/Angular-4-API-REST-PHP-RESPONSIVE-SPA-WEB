La presente aplicación te permitirá gestionar tus productos a través de una serie de funcionalidades que resumo a continuación:

  • Añadir tus propios productos de manera sencilla
  • Añadir una imagen destacada para cada producto
  • Ver el listado de todos los añadidos
  • Eliminarlos
  • Editar sus características
  • Todo ello a través de una API REST desarrollada en PHP
